# Gren Lineage Atlas — Public Viewer (v0)

A free, static website to share a **privacy‑safe, narrative genealogy**: family tree with click‑to‑open profiles and an embedded TimelineJS.

## How to publish (GitHub Pages)
1) Create a new public repo named `gren-lineage-atlas`.
2) Upload this folder’s contents to the repo root.
3) In repo Settings → Pages → Build from Branch → `main` / `/root`.
4) Your site will be live at: `https://<your-username>.github.io/gren-lineage-atlas/`.

## Update data
- Edit `data/people.json`, `data/events.json`, `data/rels.json` (safe, public subset only).
- (Optional) Replace `timeline.html` source key with your published TimelineJS Google Sheet key.

## Privacy rules
Living persons are displayed as **Private (Living)** with fields redacted. Do not include addresses or sensitive data.
